Senior Project: Malware Feature Vectorization & CNN Detection

Author: Your Name – B.Sc. Computer Science, Class of 2025

Overview

This project builds a pipeline to extract features from Android APKs, reduce their cardinality, and convert them into normalized NumPy vectors suitable for training a convolutional neural network (CNN) malware detector.

Key Components

  • de>ReduceCardinality.py – categorizes and filters features.
  • Vectorization.py – builds vector datasets from feature files.
  • de>normalize_vectors.py – optional L2/min–max/standard scaling.

How It Works

  1. Extract static features from APKs into text files.
  2. Reduce feature cardinality and write unique feature dictionaries.
  3. Vectorize APKs into NumPy arrays and feed them to a CNN.
